# MyoroPlayerMobile
MyoroPlayerMobile is the Android equivalent of https://www.github.com/antonkoetzler/MyoroPlayer using React Native

# Development
## Install React Native and setup project
`npm install --location=global react-native`
`npx react-native init MyoroPlayer`
## Running MyoroPlayerMobile
`npx react-native start`
`npx react-native run-android`
## Installing React Native navigation
`npm install @react-navigation/native @react-navigation/native-stack --save-dev`
`npm install react-native-screens react-native-safe-area-context --save-dev`
`npm install react-navigation --save-dev`
